# PCB

The clock uses a custom PCB designed for this project to connect the Arduino Nano, Max7219 display modules, and DS3231 RTC.

**Important:** The PCB design has a known flaw—an incorrect path/trace—which was only found after soldering and testing. The Gerber file is provided for reference but has this defect; builders should be aware before using it.

- **Gerber:** [Gerber_arduino_nano_clock_PCB_arduino_nano_clock_3_2025-10-06.zip](Gerber_arduino_nano_clock_PCB_arduino_nano_clock_3_2025-10-06.zip) – can be sent to a PCB manufacturer as-is (with the above caveat).
